## 项目介绍
> 本项目为 https://github.com/hequan2017/seal 项目 前端vue版本
> 基于  iview-admin 2.5.0

## Getting started

```bush
# clone the project
git clone https://github.com/hequan2017/seal


#  后端连接地址

src/config/index.js
baseUrl:
    dev:


// install dependencies
npm install

// develop
npm run dev
```

## Build
```bush
npm run build
```
